# 点击导航栏图标重启标签功能

## 🎯 功能说明

当用户点击左侧导航栏中**已选中**的网页应用图标时，自动重启该标签页。

## 💡 交互逻辑

```
第一次点击图标 → 打开/切换到该标签页
再次点击同一图标（已选中状态）→ 重启该标签页
```

## 🏗️ 实现原理

### 1. 检测已选中状态

在 `NavView_ItemInvoked` 中判断：

```csharp
// 检查是否点击的是当前已选中的项
bool isAlreadySelected = NavView.SelectedItem == args.InvokedItemContainer;

if (isAlreadySelected)
{
    // 已选中，触发重启
    WebAppRestartRequested?.Invoke(this, shortcutId);
}
else
{
    // 首次选中，正常导航
    NavigationRequested?.Invoke(this, new NavigationRequest(typeof(WebBrowserPage), shortcut));
}
```

### 2. 事件流

```
用户点击已选中的图标
    ↓
NavigationBar.NavView_ItemInvoked
    ↓
检测到 isAlreadySelected = true
    ↓
触发 WebAppRestartRequested 事件
    ↓
Linker.OnWebAppRestartRequested
    ↓
ContentArea.RestartCurrentTabAsync
    ↓
销毁旧 WebView + 创建新实例
```

## 📁 修改的文件

### 1. 导航栏 (NavigationBar.xaml.cs)

**添加事件**:
```csharp
public event EventHandler<string>? WebAppRestartRequested;
```

**修改 NavView_ItemInvoked**:
```csharp
if (tagText.StartsWith("webapp:"))
{
    string shortcutId = tagText["webapp:".Length..];
    if (_webShortcuts.TryGetValue(shortcutId, out WebAppShortcut? shortcut))
    {
        // 检查是否点击的是当前已选中的项
        bool isAlreadySelected = NavView.SelectedItem == args.InvokedItemContainer;
        
        NavView.SelectedItem = args.InvokedItemContainer;
        
        if (isAlreadySelected)
        {
            // 已选中，触发重启
            WebAppRestartRequested?.Invoke(this, shortcutId);
        }
        else
        {
            // 首次选中，正常导航
            NavigationRequested?.Invoke(this, new NavigationRequest(typeof(WebBrowserPage), shortcut));
        }
    }
    return;
}
```

### 2. 链接器 (Linker.xaml.cs)

**订阅事件**:
```csharp
NavBar.WebAppRestartRequested += OnWebAppRestartRequested;
```

**事件处理**:
```csharp
private async void OnWebAppRestartRequested(object? sender, string shortcutId)
{
    System.Diagnostics.Debug.WriteLine($"[Linker] 收到重启请求: {shortcutId}");
    await ContentHost.RestartCurrentTabAsync();
}
```

## 🐛 潜在问题与解决

### 问题1: 快速连续点击导致多次重启

**现象**: 用户快速点击图标，触发多次重启

**解决**: 添加防抖逻辑

```csharp
private DateTime _lastRestartTime = DateTime.MinValue;
private const int RestartDebounceMs = 1000;

if (isAlreadySelected)
{
    var now = DateTime.Now;
    if ((now - _lastRestartTime).TotalMilliseconds < RestartDebounceMs)
    {
        return; // 忽略过快的重复点击
    }
    _lastRestartTime = now;
    
    WebAppRestartRequested?.Invoke(this, shortcutId);
}
```

### 问题2: 重启过程中用户切换标签

**现象**: 重启A时，用户点击B，导致混乱

**解决**: 在 RestartCurrentTabAsync 中检查当前页面

```csharp
// 已在实现中处理：只重启当前显示的页面
if (_currentPage is not WebBrowserPage currentWebBrowserPage)
{
    return;
}
```

### 问题3: 用户误操作

**现象**: 用户想切换标签，但因为已选中而触发重启

**解决**: 
- 方案A: 添加确认对话框（不推荐，影响体验）
- 方案B: 添加 Tooltip 提示"再次点击重启"
- 方案C: 保持当前行为（推荐，用户会快速学习）
