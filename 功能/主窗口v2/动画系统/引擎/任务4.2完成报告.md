**实现内容**:
```csharp
private async Task AnimateWithEasing(
    WindowVisualState from,
    WindowVisualState to,
    AnimationSpec spec,
    Action<WindowVisualState> onProgress,
    CancellationToken cancellationToken)
{
    // 使用 Stopwatch 实现时间驱动（而非帧驱动）
    var stopwatch = Stopwatch.StartNew();
    var duration = spec.Duration;

    while (true)
    {
        // 检查取消令牌
        cancellationToken.ThrowIfCancellationRequested();

        // 根据真实时间计算进度（而非帧数）
        var elapsed = stopwatch.Elapsed;
        var progress = Math.Min(elapsed.TotalMilliseconds / duration.TotalMilliseconds, 1.0);
        
        // 应用缓动函数
        var easedProgress = ApplyEasing(progress, spec.Easing);

        // 插值所有属性
        var current = Lerp(from, to, easedProgress);
        onProgress(current);

        // 动画完成
        if (progress >= 1.0) break;

        // 等待下一帧（约 60 FPS）
        await Task.Delay(16, cancellationToken);
    }
}
```

**验证**: ✅ 方法已实现，包含完整的动画循环逻辑

### 2. 支持 linear、EaseIn、EaseOut、EaseInOut 缓动函数 ✅

**位置**: `功能/主窗口v2/动画系统/引擎/动画引擎.cs` 第 203-218 行

**实现内容**:
```csharp
private double ApplyEasing(double t, Easing easing)
{
    return easing switch
    {
        Easing.Linear => t,
        Easing.EaseIn => t * t,
        Easing.EaseOut => t * (2 - t),
        Easing.EaseInOut => t < 0.5 ? 2 * t * t : -1 + (4 - 2 * t) * t,
        Easing.EaseInCubic => t * t * t,
        Easing.EaseOutCubic => (--t) * t * t + 1,
        Easing.EaseInOutCubic => t < 0.5 ? 4 * t * t * t : (t - 1) * (2 * t - 2) * (2 * t - 2) + 1,
        _ => t
    };
}
```

**验证**: ✅ 所有必需的缓动函数已实现
- Linear: 线性插值（t）
- EaseIn: 二次缓入（t * t）
- EaseOut: 二次缓出（t * (2 - t)）
- EaseInOut: 二次缓入缓出（分段函数）
- 额外支持: EaseInCubic, EaseOutCubic, EaseInOutCubic（三次缓动）

## 附录：缓动函数数学公式

### Linear（线性）
```
f(t) = t
```

### EaseIn（二次缓入）
```
f(t) = t²
```

### EaseOut（二次缓出）
```
f(t) = t(2 - t) = 2t - t²
```

### EaseInOut（二次缓入缓出）
```
f(t) = {
    2t²           if t < 0.5
    -1 + (4 - 2t)t  if t ≥ 0.5
}
```

### EaseInCubic（三次缓入）
```
f(t) = t³
```

### EaseOutCubic（三次缓出）
```
f(t) = (t - 1)³ + 1
```

### EaseInOutCubic（三次缓入缓出）
```
f(t) = {
    4t³                      if t < 0.5
    (t - 1)(2t - 2)(2t - 2) + 1  if t ≥ 0.5
}
```
